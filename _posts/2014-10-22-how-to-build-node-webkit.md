Kita上已经布置好了node-webkit开发环境，可以参考一下sample-projects里面的格式，写类似的nw应用程序。
sample下载
```
			https://github.com/zcbenz/nw-sample-apps
```

要编译，只需要cd到包含nw的目录，然后:
```
			nw .
```

如果提示无法找到nw指令，来个这个吧
```
		# alias to nw
		alias nw="/Applications/node-webkit.app/Contents/MacOS/node-webkit"
```
